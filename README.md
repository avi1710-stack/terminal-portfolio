# 🖥 Terminal Portfolio

An interactive **terminal-style developer portfolio** built using JavaScript and xterm.js.
The website behaves like a real command-line interface where users can explore projects, skills, and system information through commands.

---

## 🚀 Features

* Interactive terminal interface
* Command-based navigation
* ASCII banner on startup
* Matrix animation mode
* System information (`neofetch`)

---

## 📜 Available Commands

| Command    | Description                     |
| ---------- | ------------------------------- |
| `help`     | Show all available commands     |
| `about`    | Information about the developer |
| `projects` | List of projects                |
| `github`   | Opens GitHub profile            |
| `clear`    | Clears the terminal             |
| `matrix`   | Starts matrix animation         |
| `neofetch` | Displays system information     |
